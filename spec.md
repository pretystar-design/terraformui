# Project Specification: HCL Generator

## Overview
This repository implements a minimal **HCL (HashiCorp Configuration Language) generator**. It transforms a JSON‑encoded *project model* into Terraform‑compatible HCL resource definitions. Two implementations are provided:
- **Python** (`generate_hcl.py`)
- **Go** (`generator.go` + `main.go`)

Both read a JSON file describing *nodes* (resources) and *edges* (relationships). The current implementation only renders resources; edges are noted for future expansion.

---

## Data Model (`model.go`)
```go
type Node struct {
    ID         string            `json:"id"`
    Type       string            `json:"type"`
    Attributes map[string]string `json:"attributes"`
}

type Edge struct {
    Source string `json:"source"`
    Target string `json:"target"`
    Type   string `json:"type"`
}

type Project struct {
    Nodes []Node `json:"nodes"`
    Edges []Edge `json:"edges"`
}
```
- **Node**: Represents a Terraform resource (`type`) with a unique identifier (`id`) and a map of attribute key‑value pairs.
- **Edge**: Intended to capture relationships (e.g., `depends_on`). Currently unused in generation logic but reserved for future work.

## Generation Logic
### Go (`generator.go`)
```go
func GenerateHCL(p Project) (string, error) {
    var buf bytes.Buffer
    for _, n := range p.Nodes {
        fmt.Fprintf(&buf, "resource \"%s\" \"%s\" {\n", n.Type, n.ID)
        for k, v := range n.Attributes {
            fmt.Fprintf(&buf, "  %s = \"%s\"\n", k, v)
        }
        fmt.Fprintln(&buf, "}")
        fmt.Fprintln(&buf)
    }
    return buf.String(), nil
}
```
- Iterates over `Nodes`.
- Writes a `resource "type" "id" { … }` block.
- Attributes are emitted as quoted strings.
- Edges are intentionally ignored – a **TODO**.

### Python (`generate_hcl.py`)
```python
def generate_hcl(project):
    lines = []
    for node in project.get('nodes', []):
        res_type = node.get('type')
        res_id = node.get('id')
        lines.append(f'resource "{res_type}" "{res_id}" {{')
        for k, v in node.get('attributes', {}).items():
            lines.append(f'  {k} = "{v}"')
        lines.append('}')
        lines.append('')
    return '\n'.join(lines) + '\n'
```
- Mirrors the Go implementation with straightforward string building.
- Accepts a dictionary loaded from JSON.

## Command‑Line Interface (`main.go`)
```bash
$ go run . <model.json>
```

- Prints the generated HCL to STDOUT.
- Exits with usage help when no argument is supplied.

The Python script can be used similarly:


```json
    }
  ],
  "edges": []
}
```

```hcl
resource "aws_instance" "my_instance" {
  ami = "ami-12345"
  instance_type = "t2.micro"
}
```
